Feature of cardinal and ordinal numbers. Is the number expressed by
digits, letters, or as a word?

### <a name="Word">`Word`</a>: number expressed as word

#### Examples

* _одинъ_ “one”, _двѣсти_ “200”, _тридцать_ “30”

### <a name="Digit">`Digit`</a>: number expressed using digits

#### Examples

* _1, 2, 3_

### <a name="Roman">`Roman`</a>: Roman numeral

#### Examples

* _I, II, III_

### <a name="Cyril">`Cyril`</a>: Cyrillic numeral (digit expressed using [Cyrillic letters](https://en.wikipedia.org/wiki/Cyrillic_numerals) )

#### Examples

* _РПЅ г сентябрѧ въ Ѕ де_  “in the y<ear> 186th on the sixth d<ay>“
