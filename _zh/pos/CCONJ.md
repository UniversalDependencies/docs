### Definition

A coordinating conjunction is a word that links words or larger constituents without syntactically subordinating one to the other and expresses a semantic relationship between them.

### Examples

- "and"
  - 與 / _yǔ_, 和 / _hé_, 跟 / _gēn_, 同 / _tóng_, 以及 / _yǐjí_, 並 / _bìng_, 並且 / _bìngqiě_, 而 / _ér_, 而且 / _érqiě_, 且 / _qiě_
  - Note that 與, 和, 跟, 同 may also function as prepositions (<a>`ADP`</a>)
- "or"
  - 或 / _huò_, 或者 / _huòzhě_, 還是 / _háishì_
- "but"
  - 但 / _dàn_, 但是 / _dànshì_
