### <a name="Card">`Card`</a>: cardinal numeral

#### Examples

* _<b>бір</b>_ “one”, _<b>екі</b>_ “two”, _<b>үш</b>_ “three”

### <a name="Coll">`Coll`</a>: collective numeral

Morphologically distinct class of numerals used to describe a collection of homogeneous objects.
These forms are only available for numbers 1 – 7.

#### Examples

* _<b>біреуі</b>_ “one”, _<b>екеуі</b>_ “two”, _<b>үшеуі</b>_ “three”

### <a name="Ord">`Ord`</a>: ordinal numeral

This is a subtype of adjective or adverb.

#### Examples

* _<b>бірінші</b>_ “first”, _<b>екінші</b>_ “second”, _<b>үшінші</b>_ “third”
