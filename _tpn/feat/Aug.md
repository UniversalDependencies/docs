`Augmentative` is a type of property modification in which the suffix _-(wa)usu_ is attached to a lexical root.

### <a name="Yes">`Yes`</a>: Yes


#### Examples

* _<b>kunumĩ-wasú</b>&nbsp;_ “big boy”
